축제통합관리시스템 프로젝트 소개

축제통합관리시스템은 공무원, 거래처, 국민을 위한 최적의 축제관리 서비스를 제공합니다.

관리자 화면

총괄관리자인 도 ㆍ시ㆍ군청 공무원 및 축제를 만드는 축제추진위원회가 효율적으로 모든 축제를 관리할 수 있는 시스템

국민 화면

회원가입만 하면 대한민국 모든 축제를 한곳에서 검색 할 수 있고 관심있는 축제 및 방문했던 축제를 관리를 할 수 있는 시스템

세입ㆍ세출예산 화면

경영공시로 모든 국민에게 축제에 대한 세금이 어떻게 쓰여지는 정보를 제공하는 시스템

순번 담당업무 업무내용 링크
01 로그인 권한별로 공무원,거래처, 국민, 개발사 로그인이 가능하다. 로그인 화면
02 회원가입 국민으로 회원가입을 할 수 있다.(※기업가입 구현중) 회원가입 화면
03 국민-마이페이지 국민으로 로그인시 회원정보관리, 나의 찜목록 등 확인 가능 국민-마이페이지 화면
04 관리자-마이페이지 관리자로 로그인시 관리자홈과 축제관리 가능 관리자-마이페이지 화면
05 축제등록 축제추진위원회와 거래처(위탁업체)에서 축제 등록 가능(※축제등록 구현중) 축제등록 화면
06 layout구성 관리자 화면 Thymeleaf로 layout 구성 관리자메인 화면
07 축제검색 TourAPI와 카카오지 지도 API 활용하여 위치와 축제 검색 가능 축제검색 화면
08 전체축제리스트 국민에게 진행중과 진행예정인 축제 정보 제공(※CRUD 구현중) 축제리스트 화면
09 축제 상세페이지 행사기간, 위치, 행사장소, 연락처 등 확인 가능(※CRUD 구현중) 축제 상세페이지 화면
10 계약관리 공사ㆍ용역ㆍ물품에 관련된 일반계약과 수의계약을 검색 가능(※CRUD 구현중) 계약공시 화면
11 예산관리 공무원이 관리하고 있는 경영정보를 수요자인 국민이 조회 가능(※CRUD 구현중) 경영공시 화면
12 커뮤니티 이벤트, 자주묻는 질문, 공지사항 게시글 확인 가능 커뮤니티 화면
